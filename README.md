PROYECTO: Cafetería Yossvante

Este proyecto consiste en una página web desarrollada con HTML, CSS y XML.

INSTRUCCIONES PARA ABRIR:

1. Abrir la carpeta del proyecto.
2. Buscar el archivo llamado "index.html".
3. Dar doble clic sobre el archivo.
4. La página se abrirá automáticamente en el navegador web.

Desde la barra de navegación se puede acceder a las diferentes secciones del sitio.

El archivo datos.xml puede abrirse desde el menú de navegación en la opción "Datos XML".

Proyecto realizado con fines académicos.
